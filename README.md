# 🚀 Ztooly – The Ultimate Free Online Tool Collection

&#x20;   &#x20;

## ✨ Overview

**Ztooly** is a comprehensive collection of **50+ free online tools** designed to boost productivity and simplify everyday tasks. From AI-powered generators to image editing, data conversion, PDF handling, social tools, and more — Ztooly is your all-in-one web toolkit.

> 🎯 **Mission:** Empower users with powerful, privacy-friendly tools that are fast, free, and easy to use.

---

## 🌟 Why Ztooly?

* 🆓 **100% Free** – No subscriptions, no hidden fees.
* 🔒 **Privacy-First** – Files are processed in-browser when possible.
* ⚡ **Fast & Efficient** – Optimized for performance.
* 📱 **Mobile-Optimized** – Fully responsive UI.
* 🎨 **Modern UI** – Sleek, intuitive design.
* 🔧 **Open Source** – Built for developers, extensible by anyone.

## 🚀 Getting Started

```bash
git clone https://github.com/mahfuz4223/Ztooly.git
cd Ztooly
npm install # or yarn install
```

### 🔧 Environment Setup

```bash
cp .env.example .env
```

Update `.env` as needed:

```env
VITE_APP_NAME=Ztooly
VITE_APP_URL=http://localhost:5173
VITE_ANALYTICS_ENABLED=false
```

### 🔄 Start Project

```bash
npm run dev         # Frontend
npm run server      # Backend
```

Visit [http://localhost:5173](http://localhost:5173)
